Comment faire vos recherches

Saisissez un ou plusieurs termes. Méfiez-vous des termes ambigus, tels environnement... et préférez-leur des termes plus précis comme traitement et eau. Le moteur va le(s) rechercher dans le texte du Profil et dans la pièce jointe. Attention : évitez d'utiliser toutefois le point (.) sauf pour les sigles (exemple : geide différent de g.e.i.d.e.).

Les terminaisons

Le moteur de recherche textuelle repère dans un mot la chaîne de caractères que vous avez saisie. Si vous écrivez vin, vous obtenez tous les Profils contenant les mots vin, vins ou vinicole, mais aussi celles qui contiennent le mot vingtaine, vincennes... Pour sélectionner uniquement vin, vous devez mettre le mot entre guillemets : "vin".

Les opérateurs et, ou, non

Le moteur de recherche reconnaît les opérateurs français. L'ordre des mots n'a pas d'importance sauf dans une requête contenant l'opérateur non en saisissant ingénieur et commercial, vous obtenez tous les Profils qui contiennent le mot ingénieur et le mot commercial. En saisissant ingénieur ou commercial, vous obtenez tous les Profils qui contiennent le mot ingénieur ou le mot commercial. En saisissant ingénieur non commercial, vous obtenez tous les Profils qui contiennent le mot ingénieur à l'exception de celles qui contiennent le mot commercial. Attention : vous devez utiliser l'opérateur non entre deux termes. Une requête du type non ingénieur est impossible. Si vous n'utilisez pas d'opérateur, exemple ingénieur commercial, le moteur sous-entend par défaut l'opérateur ou entre les deux mots.

Les expressions exactes

Si vous recherchez une expression précise, il est nécessaire de la saisir entre guillemets. Exemples de requête : "école de commerce" ou "grande distribution" ou "cloud".

Les parenthèses

Les opérateurs peuvent se combiner avec des parenthèses. En saisissant ingénieur et (affaires ou commercial), vous obtenez tous les Profils contenant les mots ingénieur et affaires ou les mots ingénieur et commercial. En saisissant (ingénieur et affaires) ou commercial, vous obtenez tous les Profils contenant les mots ingénieur et affaires ou le mot commercial. »
